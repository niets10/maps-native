# Expo HAS CHANGED

This project targets **Expo SDK 54** (downgraded from SDK 57 on 2026-07-16 because the App
Store/Play Store build of Expo Go only supports SDK 54 — SDK 55+ builds are stuck in app review).

Read the exact versioned docs at https://docs.expo.dev/versions/v54.0.0/ before writing any code.

Do not upgrade back to SDK 57+ packages unless Expo Go on the app stores has caught up, or you've
set up EAS development builds instead of relying on Expo Go.
